## Apple Developer Program 更新手順

1. 下記のようなメールが届きます。
   ![20230111_apple-developer-program-update_01](/docs/files/20230111_apple-developer-program-update_01.png)

2. https://developer.apple.com/account/ にアクセスします。

3. ログイン ID とパスワードを入力します。
   ![20230111_apple-developer-program-update_02](/docs/files/20230111_apple-developer-program-update_02.png)

4. ２段階認証が求められますので、末尾 51（大槻様の電話番号）の電話番号を使い認証します。
   ![20230111_apple-developer-program-update_03](/docs/files/20230111_apple-developer-program-update_03.png)

5. 「信頼する」ボタンをクリックします。※どのボタンを押しても本手続きに影響はありません。
   ![20230111_apple-developer-program-update_04](/docs/files/20230111_apple-developer-program-update_04.png)

6. 「メンバーシップを更新」ボタンをクリックします。
   ![20230111_apple-developer-program-update_05](/docs/files/20230111_apple-developer-program-update_05.png)

7. 再度、ログイン ID とパスワードを入力します。
   ![20230111_apple-developer-program-update_06](/docs/files/20230111_apple-developer-program-update_06.png)

8. 「クレジットカードまたはデビットカード」をクリックします。
   ![20230111_apple-developer-program-update_07](/docs/files/20230111_apple-developer-program-update_07.png)

9. 必要事項を記入し、「注文の確認」ボタンをクリックします。
   ![20230111_apple-developer-program-update_08](/docs/files/20230111_apple-developer-program-update_08.png)
